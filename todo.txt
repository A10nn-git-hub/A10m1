Слева от кнопки "Играть" должна быть квадртаная кнопка с пиктограммой "шестеренка". 
По ее нажатию должны отображаться настройки конкретной игры в новом окне. 
Хост должен иметь возможность менять настройки, а присоединившиеся игроки - только просмотреть. Для хоста должна быть кнопка "Сохранить".
настройки - это таблица, в которой первой колонке выводится список игроков, зашедшими в лобби. Если никто не зашел, то выводится только 1 строка - хост.
Следующие колонки для каждой игры должны быть свои. 


Настройки для игры 2D выживание:
1. вторая колонка - КОЛ-ВО жизней (тут можно поставить себе много очень жизней, или компу)
2. третья колонка -  КОЛ-ВО патронов в сек (ставишь макс 10 и можешь играть как с читами)
3. четверная колонка - только для ИИ. можно установить УРОВЕНЬ (1,2 или 3 система другой не дает поставить). Этот уровень должен влиять на логику ботов: 1 уровень: играет просто носится по карте, 2 уровень: стреляет по тебе. 3 уровень: гоняется за тобой и стреляет (самый гиперсложный)
4. ниже таблицы нужно выводить чекбокс с текстом "уменьшение поля". Если галочка установлена, то включается режим уменьшения красного круга, внутри которого можно играть. Если галочки нет, то круг не уменьшается.


Настройки для игры крестики-нолики:
1. вторая колонка - фигура, которой играет игрок (крестик, нолик, квадрат, треугольник, круг). По умолчанию у каждого игрока фигура по порядку из списка. При нажатии на эту фигуру она сменяется на следующую по списку. Если у нескольких игроков одна и та же фигура, то настройки сохранить нельзя.
2. третья колонка - цвет (зеленый, красный, синий, фиолетовый, желтый). Цвета одинаковые можно сохранять.
3. ниже таблицы нужно выводить настройку с размером поля: от 3 до 10. при указании числа поле должно формироваться как квадрат со указанной стороной
